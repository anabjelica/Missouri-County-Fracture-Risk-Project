# Osteoporosis-Risk-Prediction-and-Model-Evaluation
# Project Overview:
# This project analyzes Missouri county-level data to examine fracture risk related to low bone density and socioeconomic and healthcare risk factors. Twelve variables were standardized and combined into a composite Unmet Risk Index (URI), and counties were clustered into three risk groups and visualized geographically. The goal is to highlight regional disparities and support earlier identification of high-risk populations.
# Methods:
# - Missouri county dataset was cleaned and organized into three domains: biological, socioeconomic, and healthcare access factors
# - Variables were standardized using z-score normalization
# - A composite risk score (URI) was created by aggregating standardized variables
# - Counties were grouped into 3 clusters based on risk level
# - Results were visualized using statistical plots and geographic mapping
